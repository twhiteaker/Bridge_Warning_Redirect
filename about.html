About Bridge Warning Redirect

Demo

Bridge Warning Redirect on GitHub

This project demonstrates how to download forecasted streamflow and forward the data to a Web app that plots a time-enabled cross section showing bridges and water levels.

Data Sources

Streamflow data originate from the NOAA's National Water Model (NWM). The data are read using the National Water Prediction Service (NWPS) API. This service provides access to the latest short range forecast.

Bridge data originate from LiDAR data in Texas, available from TxGIO. Data were analyzed to determine bridge locations and ground surface elevations, with rough estimates used for deck thickness.

Usage

This Web app accepts these inputs:

  • Bridge ID - A unique identifier assigned to a bridge upon extraction from LiDAR data.
  • Stream ID - The identifier for the National Water Model stream reach for which streamflow should be downloaded

Use the input form and click Plot to retrieve the latest forecast and forward the data to the cross section plotter. The plotter is a separate application not maintained in this repository or by this repository's owner.
